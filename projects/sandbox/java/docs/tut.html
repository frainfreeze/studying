FAQ

Q: log4j cant find configuration but it is there A: got to java build path in project properties and make sure src/main/resources has excluded set to none

Q: maven test: log4j cant find configuration but it is there A: Set resources in pom.xml

Q: No compiler is provided in this environment. Perhaps you are running on a JRE rather than a JDK? A: install new jre and jdk
