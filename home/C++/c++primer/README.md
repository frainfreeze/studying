## C++ Primer 5 (C++11/14)

### Compilers recommended
 * Windows: Visual Studio 2015+ 
 
 * Linux: g++ 5.0+     
```bash
g++ -std=c++14 some_ex.cpp -o some_ex
```
 
 * Mac: clang++ 3.7+
```bash
clang++ -std=c++1y some_ex.cpp -o some_ex
```
